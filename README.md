# 🌐 Jishnu MS Portfolio

A modern and responsive developer portfolio built using React + Vite.  
It showcases my projects, skills, and includes a working contact form.

🚀 Live Demo: https://jishnu-ms.vercel.app/

## ✨ Features

- 🎨 Modern responsive UI
- ⚡ Built with React + Vite
- 📬 Contact form with Web3Forms
- 🧭 Smooth scrolling layout
- 💼 Project showcase section
- 🔗 Social media integration
- 🚀 Deployed on Vercel

## 🛠️ Tech Stack

- ⚛️ React
- ⚡ Vite
- 🟨 JavaScript
- 🎨 CSS
- 🎯 React Icons
- 📩 Web3Forms API
- ▲ Vercel

## 🚀 Getting Started

Clone the repository

git clone https://github.com/Jishnu-ms/portfolio.git

Navigate to project folder

cd portfolio

Install dependencies

npm install

Run development server

npm run dev

Build for production

npm run build

## 📬 Contact Form Setup

Create a `.env` file in the root directory

VITE_WEB3FORMS_KEY=your_access_key_here

Restart server

npm run dev


## 🌍 Deployment

This project is deployed using Vercel.

Build project

npm run build

Deploy the dist folder.
